▣ 총 12 단계 ▣

01. create window
 - 윈도우 생성하기
 

02. draw line
 - 선 그리기 : 선 그리기를 이용해서 사각형을 그린다.
 

03. rect class
 - 사각형 클래스 : 사각형의 속성을 가진 클래스를 작성한다.
 

04. clear and update
 - 다시 그리기 : 윈도우에서 다시 그리기의 개념을 익힌다.
 

05. sort by size
 - 크기별 정렬 : 사각형의 크기순으로 정렬하는 알고리즘을 적용한다.
 

06. card class
 - 카드 클래스 : 카드의 속성을 정의한 클래스를 작성한다.
 

07. figure class
 - 도형 클래스 : 클래스의 상속 및 다형성의 개념을 익힌다.
 

08. card class using bitmap
 - 비트맵 카드 : 선 그리기를 이용한 카드 출력을 이미지로 변경한다.
 

09. card pile class
 - 카드 뭉치 클래스 : 카드 뭉치의 속성을 정의한 클래스를 작성한다.
 

10. create game - find couple
 - 실전 게임 만들기 : 짝찾기 게임을 만들어본다.


11. create game - solitare
 - 실전 게임 만들기 : 솔리테어 게임을 만들어본다.
 

12. create game - solitare extend
 - 실전 게임 만들기 : 솔리테어 게임에 추가 기능(배경 이미지 추가, 상태 리스트 표시, 엔딩 애니메이션 추가)을 작성한다.
